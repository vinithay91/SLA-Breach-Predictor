# AI Usage Note

This project uses AI/LLM capabilities for SLA breach prediction and analysis.

## AI Features

1. **Risk Analysis**: Uses LLM to analyze ticket data and predict SLA breach risks
2. **Report Generation**: Automated generation of breach risk reports
3. **Recommendations**: AI-powered suggestions for preventing SLA breaches

## Usage Guidelines

- Ensure API keys are properly configured in `.env` file
- Monitor token usage to manage costs
- Review AI-generated recommendations before implementation
- Use AI outputs as decision support, not as sole decision makers

## Dependencies

- OpenAI API (or compatible LLM service)
- Required libraries listed in `requirements.txt`
